
   ```bash
   flutter pub get
   ```

   ```bash
   flutter run
   ```

5. **Build APK** (Android)
   ```bash
   flutter build apk --release
   ```

6. **Build IPA** (iOS)
   ```bash
   flutter build ios --release
   ```
